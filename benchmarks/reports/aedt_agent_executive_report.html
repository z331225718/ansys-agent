4. 为什么节点化重要

自由代码路径的问题受控节点路径的改进
LLM 可能调用不存在或版本不匹配的 API节点内部封装真实 PyAEDT 调用和版本兼容
端口、边界、face selection 容易 runtime error节点有 schema、postcheck、audit 和错误归因
失败原因散落在任意 Python 代码中失败集中在节点输入、引用、validation，便于修复
安全边界弱Group C 自由代码执行次数为 0

5. 当前边界

当前 validation 是结构性判卷，不等同于完整电磁物理正确性证明。
当前能检查未来要增强
对象、材料、端口、边界、setup/sweep 是否创建电磁语义、求解结果、工程合理性
关键引用关系是否合理复杂模型、后处理、长时间任务恢复
